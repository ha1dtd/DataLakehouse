# HDOS Sample Add-on

This folder is a product-style Lakehouse add-on package scaffold based on the agreed add-on contract.

It is intentionally not a direct copy of the current `dags/hdos_widget/` implementation.

Purpose:

- show the required add-on package shape
- show how one customer package can hold one or more domains
- provide a concrete widget-oriented starting point for the first real add-on implementation

Current domain in this sample:

- `dashboard_widgets`

This scaffold is package-structure only. Runtime logic still needs to be implemented and validated against the active Lakehouse platform.
